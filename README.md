# Lodestar

Use Minecraft's Boss Bar to show player's current coordinates + compass direction facing + progress bar shows remaining amount of daylight or night.

# Building

```
mvn clean ; mvn package
```

# Install

```
cp target/Lodestar-1.0.0.jar ~/minecraft/plugins/
```
